## Getting Started

### 1. Install packages
```bash
npm i
```

### 2. Set up env vars
To get the values, please contact me! [Email](mailto:briankim00119@gmail.com)

### 3. Run the development server (local)

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### 4. See the result!
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
